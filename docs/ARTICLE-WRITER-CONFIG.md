# 2. WRITING RULES

## Voice
- Match stated voice: VOICE_STYLE
- Write for: TARGET_AUDIENCE
- Evidence-based: cite sources, name people, include dates
- Hedge appropriately: "research suggests" not "research proves"

## Data density
- ≥3 specific dates or time periods per article
- ≥3 named figures or authorities
- ≥1 primary source reference
- ≥1 evidence hedge ("suggests," "indicates")

## AEO (Answer Engine Optimization)
- First 1-2 sentences: factual answer with definitional verb
- ≥2 question-format H2/H3 headings
- ≥1 "X is Y" definition sentence per section
- ≥1 bulleted or numbered list

## Banned phrases
```
unlock, game-changer, delve, realm, dive in,
furthermore, in conclusion, it's important to note
```

## Word count targets
- Hub articles: 2000+ words
- Regular articles: 1200+ words
